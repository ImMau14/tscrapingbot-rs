# Contributor Covenant Code of Conduct (v2.1)

This project follows the Contributor Covenant Code of Conduct v2.1.
All contributors are expected to abide by it.

To report unacceptable behavior, open a private issue with the title `Code of Conduct Report` or contact the repository owner via GitHub.

You can read the full Contributor Covenant at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
